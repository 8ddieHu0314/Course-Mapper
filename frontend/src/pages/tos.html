Terms of Service

By using this tool, you agree that:

  • You will not rely on this site as your sole source of schedule or navigation information.
  • You understand that the developer is not liable for missed classes or delays.

Privacy

The tool does not require account creation or store personal data by default. If hosted on a platform that logs requests (e.g., a standard web server), basic technical metadata such as IP address and timestamps may be recorded by that platform.

FAQ

How do I add my real courses and buildings?

Open index.html and edit the COURSE_DATA array in the script. Each course has a name, lat, and lng. Replace the placeholder entries with real building coordinates from your campus.

Can I change the order of the route?

Yes. The route follows the order of the courses you select. Select them in the order you attend them (e.g., 9:00 class → 10:10 class → 11:25 class).

Does this use actual walking paths?

Yes, the map uses Google’s walking directions between points. It’s still a good idea to verify the paths and timing yourself, especially if your campus has shortcuts that aren’t mapped.

Can I deploy this somewhere?

You can host this as a static site (e.g., GitHub Pages, Netlify, Vercel, or your school’s servers). Just remember to keep your Google Maps API key secure and restrict it in the Google Cloud Console (HTTP referrer restrictions).
